m-type

responsive typography

is a tool to easily and quickly set-up typography for the web: provided with a few values, it will automatically generate a baseline grid and a modular scale, producing headings in proportional sizes and a body of text well organized and balanced.

It's also possible to set alternative layouts for different devices and screen sizes. Just resize this window to see the typography change, adapting to the available space.
